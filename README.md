# Bee Colony Data Science Pipeline 🐝

End-to-end data science project analyzing US Bee Colony 
Census data across counties and states.

## What it does
- Full data science lifecycle on real USDA bee colony dataset
- EDA to surface patterns across states, years, districts
- Classification: predict if colony value is above/below median
- Regression: predict exact colony count values

## Dataset
Kaggle — Bee Colony Census and Loss Data
- Features: Year, State, County, Agricultural District, Value
- Source: USDA county-level bee colony observations

## Tech Stack
Python, Pandas, Scikit-learn, Matplotlib, 
Seaborn, Statsmodels

## Pipeline Stages
1. Problem Understanding & Dataset Selection
2. Exploratory Data Analysis (EDA)
3. Data Preprocessing & Feature Engineering
4. Model Training (Classification + Regression)
5. Model Evaluation
6. Deployment

## How to run
pip install pandas scikit-learn matplotlib seaborn statsmodels
jupyter notebook DS_project.ipynb
